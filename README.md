printgic
=================

PrintGic is a Node.js web framework.

Installation
-----------
1. In root directory run command
``` sh
$ npm install
```
2. Go to .core directory run command
``` sh
$ npm install
```
3. Start application
``` sh
# Windows
$ set DEBUG=printgic:*
$ npm run debug
```
``` sh
# Mac or Linux
$ npm run dev
```
``` sh
# Normal mode
$ npm start
```
4. Generate API document
```
$ npm run doc
```
